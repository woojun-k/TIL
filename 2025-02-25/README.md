## Web

### Web의 구성

- HTML
- CSS
- JS

#### HTML 구성요소

- 기본 구조
    - `<html\>` `</html>`
    - `<body>` `</body>`

- 요소와 태그
    - HTML의 요소는 태그와 내용으로 구성
    - 내용이 없는 태그들
    - 요소는 중첩될 수 있음
    - 여는 태그와 닫는 태그의 쌍을 잘 확인해야 함

#### DOM 트리

- HTML문서를 브라우저에 렌더링 하기 위한 구조'
- 계층적 구조(부모-자식)로 문서를 표현하여 요소 관계가 명확
    - 조상관계 - ancestor
    - 자손관계 - descendant
    - 부모관계 - parent children
    - 형제관계 - sibling
- 모든 HTML 요소, 속성, 텍스트는 각각 하나의 노드로 취급
